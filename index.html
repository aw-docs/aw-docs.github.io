The official Altos Wars documentation

Welcome to the official documentation for the minecraft custom game Altos Wars! Altos Wars is a 1v1 pvp game, assigning both players to either team Hunters or team Altos in an epic battle to destroy each other's cores with the help of unique items acquired through killing various mobs. Altos Wars initially began as a joke, but has now become such a large project that it is worth creating docs for. These docs are split into various sections which you can find below. To stay on top of updates to the game, check out the patch notes. Please note that this site is still under construction as many pages are missing and many documents are poorly written. Download links for Altos Wars save files will be provided soon.
